# OverDrive! Music Platform
---
* 主要內容：
  * 資料、視覺、功能流程方面參考 HUF 及 iNDIEVOX 等網站
  * 基本註冊登入功能
  * 使用 nodemailer 實現回信功能
  * 利用 Google Map API 呈現場館位置
  * 利用 react-i18next 實現中英網站內容切換
  * 了解 env 檔，保護敏感資訊
* 使用技術：
  * HTML
  * SCSS
  * Bootstrap
  * react-bootstrap
  * react-icon
  * React.js
  * nodemailer
  * Google API
  * react-i18next
  * Node.js
  * MySQL
  * Git
